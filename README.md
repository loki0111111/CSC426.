## 1. GUI Calculator

A simple, user-friendly browser-based calculator built with HTML, CSS, and JavaScript.

### Features
- Basic arithmetic: addition, subtraction, multiplication, division
- Integer division (`\`)
- Exponentiation (`^`)
- Modulus (`%`)
- Clear (`C`) to reset the display
- Expression preview shown above the result
- Keyboard support
- Input length capped at 15 characters to prevent overflow

### How to Run
Open `calculator/calculator.html` in any web browser. No installation required.

## 2. Login Authentication App

A web-based login authentication application built with HTML, CSS, and JavaScript. Deployed on Vercel.

### Features
- Username and password input fields
- Login (Submit) and Reset (Cancel) buttons
- Input validation — checks for empty fields before submission
- Success message on correct credentials
- Error message on invalid or missing credentials
- Fields highlighted in red on validation failure
- Responsive user interface
